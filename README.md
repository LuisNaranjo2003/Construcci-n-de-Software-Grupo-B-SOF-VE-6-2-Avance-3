# BOOKORA_EC - Sistema de Gestión de Reservas para Peluquería

## Descripción del Proyecto
BOOKORA_EC es un sistema web desarrollado para la gestión de reservas y servicios de una peluquería.  
El sistema permite administrar citas, usuarios, servicios y horarios de atención de manera organizada y eficiente.
